# Nate Prod Editor

A web-based documentation/table editor with local autosave and GitHub integration.

## Features
- Edit and manage documentation tables in your browser
- Autosave to local storage (opt-in, toggle in the UI)
- Download your work as a JSON+SQL zip package
- Submit changes directly to GitHub via Pull Request

## Development
- All code is in the `js/`, `css/`, and root HTML files
- No build tools required for basic use
- For advanced features, see comments in the code
